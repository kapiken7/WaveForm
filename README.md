# WaveForm Monitor - 時系列データ監視・解析アプリケーション

## 概要
  <br>リアルタイムで生成される時系列データ（波形）をモニタリングし、統計解析と設定されたしきい値に基づく異常検知を行うWindows Formsアプリケーションです。

---

## 機能

* **リアルタイムデータ生成とグラフ表示**
    <br>架空の2バイトバイナリデータ（0〜65535）を一定間隔で生成し、時系列グラフにリアルタイムで描画します。
* **データ制御と設定**
    <br>データの**更新間隔**（ミリ秒）と**異常検知のためのしきい値**をUIから設定・変更可能。設定値は次回起動時に自動復元されます。
* **データ解析と統計表示**
    <br>直近の**20件**のデータに基づき、現在値、**平均値**、**最大値**、**最小値**をリアルタイムで表示します。
* **しきい値判定とアラート:**
    <br>最新のデータがしきい値を超えた場合、**「異常」**としてUIにアラート表示されます。「判定リセット」ボタンでアラートをクリアできます。
* **ログ記録:**
    <br>すべてのデータと解析結果を日ごとの「`DataLog_YYYYMMDD.csv`」に、アラート状態の遷移（正常⇔異常）を「`AlertLog_YYYYMMDD.csv`」に記録します。（UTF-8 BOM付き）

## ビルド方法

本プログラムはVisual Studioで開発しており、以下の手順でビルドできます。

1.  プロジェクトをVisual Studio 2026で開きます。
2.  メニューバーの **「ビルド」** から **「ソリューションのビルド」** を選択します。
3.  実行ファイル（`.exe`）は `bin/Debug/net10.0-windows/` などのフォルダに出力されます。

---

## 使用方法

1.  実行ファイルを起動します。
2.  設定エリアで、**「インターバル(ms)」**と**「しきい値」**を設定します。
3.  **「Start」** ボタンをクリックし、モニタリングを開始します。
4.  グラフと統計情報がリアルタイムで更新されます。
5.  最新データがしきい値を超えた場合、アラートが表示されます。**「判定リセット」** ボタンでアラートをクリアできます。
6.  **「Stop」** ボタンでモニタリングを停止します。
7.  アプリケーションを終了すると、設定エリアの設定が保存されます。
